# Repository directory

This directory holds the published XBPS binary packages and repository
metadata. Contents are generated by `scripts/build.sh`.

## Files

- `*.xbps` — Package archives (generated by xbps-src)
- `*.sig`  — Ed25519 signature over the repository index (generated by
  `scripts/sign-repo.sh`)
- `*-repodata` — XBPS repository index files

## Usage

This directory is served as an XBPS repository. On a Void Linux system:

```bash
# Add the repository (adjust URL for your hosting method)
xbps-install -R "https://github.com/your-org/void-repo/releases/latest/download" \
  -r /tmp/vr --sync

# Install a package
xbps-install -r /tmp/vr <pkgname>
```

See the main README for complete instructions.
